# Open RPG
This is an HTML5 web app for creating (and sharing) RPGs

## Features
* Git-friendly development
* 2.5d gameplay
* Dynamic resource handling
* Level-building tool
  * Tileset editor
  * Prop editor (TODO)
  * Level preview (TODO)
* NPC dialog tool (TODO)

##  To install
`$ pip install -r requirements.txt`

## To run
`$ make`

## To generate documentation
`$ make documentation`
